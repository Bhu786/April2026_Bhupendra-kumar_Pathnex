# 🔐 AWS Security, Networking & DevOps Notes

---

# 🛡️ Security Tools

## 🔹 AWS Shield

**Definition:**
Managed **DDoS protection service**.

**Use Case / Practical:**

* Protect websites & APIs from traffic flooding attacks
* Works automatically with CloudFront & ALB

👉 **Interview Point:**
Q: What tool is used for DDoS protection?
👉 Answer: **AWS Shield + AWS WAF**

---

## 🔹 AWS WAF (Web Application Firewall)

**Definition:**
Firewall to protect web apps from common attacks.

**Use Case / Practical:**

* Block SQL Injection
* Block XSS attacks
* Restrict IPs

---

## 🔹 Amazon GuardDuty

**Definition:**
Threat detection service using ML.

**Use Case / Practical:**

* Detect suspicious activity
* Identify compromised EC2 or IAM

---

## 🔹 Amazon Inspector

**Definition:**
Security assessment service.

**Use Case / Practical:**

* Scan EC2 for vulnerabilities
* Identify outdated packages

---

## 🔹 AWS KMS (Key Management Service)

**Definition:**
Service to manage encryption keys.

**Use Case / Practical:**

* Encrypt data in S3, EBS
* Secure sensitive information

---

## 🔹 AWS Systems Manager (SSM)

**Definition:**
Manage and automate EC2 instances.

**Use Case / Practical:**

* Access EC2 without SSH
* Run commands remotely
* Store parameters securely

---

## 🔹 Encryption Types

### Symmetric Key

* Same key for encryption & decryption
* Faster

### Asymmetric Key

* Public key + Private key
* More secure
