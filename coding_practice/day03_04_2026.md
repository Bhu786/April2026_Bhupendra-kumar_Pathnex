# 📘 Linux / CLI Notes

## 📂 File Extensions

In CLI/Linux, extensions don’t matter:

```
hello.pdf == hello.txt == hello
```

👉 Extensions are just for our understanding.

---

## 🔐 File Permissions

```
rwx ==> read, write, execute
r = 4
w = 2
x = 1
```

### Structure:

```
OWNER   GROUP   OTHERS
-rw-    ---     ---
```

### Permission Values:

* 7 = r + w + x = 4 + 2 + 1
* 6 = r + w = 4 + 2
* 5 = r + x = 4 + 1
* 4 = r only

---

## 📌 Important Examples

| Permission | Meaning                            |
| ---------- | ---------------------------------- |
| 777        | Everyone full access               |
| 755        | Owner full, others read + execute  |
| 644        | Owner read/write, others read only |

---

## ⚙️ Default Permissions

* Directory → `777`
* File → `666`
* Umask → `022` ⭐ (Important for interview)

---

## 🔧 chmod Commands

### Give execute permission:

```
chmod +x FILE_NAME
```

### Example:

```
chmod 755 FILE_NAME
```

👉 Meaning:

* Owner → rwx
* Group → r-x
* Others → r-x

## 👤 User Management

### Switch user:

```
su
```

### Create new user:

```
useradd USER_NAME
passwd USER_NAME
```

### Exit user:

```
exit
```

### Add user to sudo group:

```
sudo usermod -aG sudo USER_NAME
```

### Avoid typing sudo again & again:

```
sudo -i
```

---

## 👥 Group Management

### Create group:

```
groupadd GROUP_NAME
```

### Change group of file:

```
chgrp GROUP_NAME FILE_NAME
```

---

## 📋 View Users & Groups

```
cat /etc/passwd   # View users
cat /etc/group    # View groups
```
