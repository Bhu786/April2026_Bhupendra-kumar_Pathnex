
file structure :

variable.tf: we define all input here 

terraform.tfvars : yeha varibale ki actual value dete hai 

outputs.tf : whatever output comes after terraform run 
provider.tf : which cloud u r using like aes or azure

.terraform.lock.hcl : version fix karta hai

terraform.tfstate: brain of terraform  
.terraform/ : interanl folder like plugins ,modules,backend data .

terraform init # initialize directory download all necessary file for run terraform
terraform fmt # format code
terraform validate # validate config
terraform plan # review changes
terraform apply # apply changes
terraform output # get useful info from outputs
terraform destroy # destroy infra when needed
