Agar tum **Git me `main` branch se pull karna chahte ho**, toh simple command hai:

```bash
git pull origin main
```

### Samajh lo simple words me:

* `git pull` → latest changes le aata hai
* `origin` → remote repo (GitHub wala)
* `main` → jis branch se pull karna hai

---

### Agar tum kisi aur branch par ho (jaise `dev`) aur `main` ka code lana hai:

```bash
git pull origin main
```

Ya better practice:

```bash
git fetch origin
git merge origin/main
```

---

### Check kar lo tum kis branch par ho:

```bash
git branch
```

---

Agar tum bata do exact scenario (merge karna hai, rebase karna hai, ya conflict aa raha hai), toh main uske according perfect command bata dunga 👍
